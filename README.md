# 比赛简易评分系统

基于 Anaconda + PyCharm 实现的比赛评分系统，满足任务要求的全部功能。

## 功能特点

1. **用户验证功能**
   - 支持多用户登录（admin/judge/user）
   - 密码加密存储（SHA256）
   - 3次登录失败限制

2. **评分功能**
   - 设置评委人数和姓名
   - 设置选手人数和姓名
   - 自动去掉最高分和最低分
   - 按平均分排序排名
   - 处理并列名次情况

3. **文件管理**
   - 支持多种格式保存（CSV/Excel/JSON）
   - 自动生成时间戳文件名
   - 查看历史结果文件

4. **用户管理**
   - 管理员可添加新用户
   - 角色权限控制

## 环境要求

- Python 3.7+
- Anaconda 环境（推荐）
- 以下Python包：
  - pandas
  - numpy
  - openpyxl（用于Excel支持）
  - colorama（用于彩色输出）
